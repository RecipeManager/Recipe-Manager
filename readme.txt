== Description ==

Recipe Manager is a lightweight recipe plugin which allows you to display recipes on your WordPress website.

The recipe editor was designed with usability and flexibility in mind. It can handle basic & complex recipes in a simple way.

Create recipes with multiple phases (Cake Batter, Frosting, Filling), each with its own ingredients & steps.

=== Features ===

- Adds a recipe custom post type (CPT) to your website
- Featured image for each recipe
- Recipe categories & tags
- Recipe details, ingredients & instructions
- Recipe preparation and cooking times
- Recipe servings
- Custom recipe course assignment (e.g Starter, Main, Dessert)
- Unlimited ingredients
- Ingredient information (Allergenic, Dairy)
- Mobile responsive
- Display with easy [recipe] & [recipes] shortcodes
- schema.org Markup - Works with Google Rich Snippets
